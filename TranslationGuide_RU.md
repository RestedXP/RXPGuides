# Руководство по переводу гайдов RestedXP на русский язык

Это руководство предназначено для переводчиков и ИИ, помогающих в локализации аддона RestedXP Guides.

## Структура файлов
- Все русские гайды должны находиться в директории `lang/Guides-ruRU/`.
- Файлы должны быть разделены по дополнениям (например, `lang/Guides-ruRU/tbc/`).
- Каждый файл гайда должен быть зарегистрирован в соответствующем XML-списке (например, `lang/Guides-ruRU/GuideList-tbc.xml`).

## Правила перевода для ИИ

При переводе гайда следуйте этим инструкциям:

1. **Заголовок файла**:
   - Начинайте файл с проверки локали: `if GetLocale() ~= "ruRU" then return end`.
   - Переводите название группы (`#group`) и название гайда (`#name` / `#title`).

2. **Команды гайда (теги)**:
   - **НЕ ПЕРЕВОДИТЕ** ключи команд, такие как `.goto`, `.accept`, `.turnin`, `.target`, `.mob`, `.use`, `.xp`, `.subzone`, `.zone`, `.isOnQuest`.
   - **НЕ ПЕРЕВОДИТЕ** названия локаций в командах (например, в `.goto` или `.zone`), так как они используются как внутренние идентификаторы (например, `.goto Deadwind Pass,42.88,34.52`).
   - **ПЕРЕВОДИТЕ** текст после разделителя `>>`.
   - **ПЕРЕВОДИТЕ** названия NPC в тегах `.target` и `.mob`.
   - **ПЕРЕВОДИТЕ** названия предметов в квадратных скобках внутри текста.
   - Сохраняйте коды цветов WoW (например, `|cRXP_WARN_...|r`) и иконки (например, `|T...|t`).

3. **Пример перевода**:

**Оригинал (EN):**
```lua
step
    .goto Deadwind Pass,47.0,75.6
    >>|Tinterface/worldmap/chatbubble_64grey.blp:20|tTalk to |cRXP_FRIENDLY_Archmage Alturus|r
    .accept 9824 >> Accept Arcane Disturbances
    .target Archmage Alturus
```

**Перевод (RU):**
```lua
step
    .goto Deadwind Pass,47.0,75.6
    >>|Tinterface/worldmap/chatbubble_64grey.blp:20|tПоговорите с |cRXP_FRIENDLY_Верховным магом Альтурусом|r
    .accept 9824 >> Примите задание "Магические возмущения"
    .target Верховный маг Альтурус
```

## Промпт для ИИ переводчика

Используйте следующий промпт для перевода гайда:

"""
Ты — профессиональный переводчик игрового контента World of Warcraft. Тебе нужно перевести гайд аддона RestedXP с английского на русский.

ИНСТРУКЦИИ:
1. Оставь `if GetLocale() ~= "ruRU" then return end` в начале.
2. Переведи значения тегов `#group`, `#name`, `#title`.
3. В строках с `>>` переведи только текст после `>>`. Сохраняй все коды цветов `|c...|r` и иконки `|T...|t`.
4. В командах `.accept`, `.turnin` переведи названия заданий после `>>`.
5. В командах `.target` и `.mob` переведи имена NPC/монстров на русский язык.
6. В командах `.goto`, `.zone`, `.subzone` **НЕ ПЕРЕВОДИ** название локации (первый аргумент).
7. Сохраняй структуру Lua-кода и все идентификаторы (ID заданий, предметов).

ПЕРЕВЕДИ СЛЕДУЮЩИЙ КОД:
[ВСТАВИТЬ КОД ГАЙДА ЗДЕСЬ]
"""

## Регистрация нового гайда
После создания файла гайда:
1. Добавьте `<Script file="путь\к\файлу.lua"/>` в соответствующий XML в `lang/Guides-ruRU/`.
2. В оригинальном английском гайде добавьте проверку: `if GetLocale() == "zhCN" or GetLocale() == "ruRU" then return end`, чтобы избежать дублирования.
